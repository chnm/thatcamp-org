» Installation Instructions
  1. Open wp-content/plugins Folder
  2. Put:
    Folder: wp-dbmanager
  3. Activate WP-DBManager Plugin
  4. Open wp-content/backup-db Folder
  5. The script will automatically create a folder called backup-db in the wp-content folder if that folder is writable. If it is not created, please create it and CHMOD it to 777
  6. Renamed htaccess.txt to .htaccess file in
    Folder: wp-content/plugins/wp-dbmanager
  7. Move the .htaccess file from:
    Folder: wp-content/plugins/wp-dbmanager
    to:
    Folder: wp-content/backup-db
  8. Refer To Usage For Further Instructions
» Upgrade Instructions
From v2.0x To v2.60
  1. Deactivate WP-DBManager Plugin
  2. Open wp-content/plugins Folder
  3. Put/Overwrite:
    Folder: wp-dbmanager
  4. Delete this folder if exists:
    Folder: dbmanager
  5. Activate WP-DBManager Plugin
  6. Refer To Usage For Further Instructions
